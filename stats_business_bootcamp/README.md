# Stats + Business Bootcamp (12 weeks)

## How to use
1) Open this folder in Jupyter / VS Code.
2) Run notebooks in order: week01 → week12.
3) In every notebook, fill **one-sentence insight** prompts and the **Executive Summary** section.

## Data
- `data/sales_marketing_example_2025.csv` : synthetic but realistic funnel-style dataset
- `data/data_dictionary.csv` : column meanings

## Recommended routine (30–45 mins/day)
- Theory bullets (5–10 mins)
- Run analysis cells (15–20 mins)
- Write interpretation + one-sentence business insight (10 mins)
- Copy Executive Summary into email/notes (5 mins)
